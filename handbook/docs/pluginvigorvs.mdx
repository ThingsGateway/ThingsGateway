## 二、变量配置

:::tip `说明`

通用格式

4字节转换格式

DATA=ABCD;

举例：

DATA=ABCD; ，代表大端格式，其中

ABCD=>Big-Endian;

BADC=>Big-Endian Byte Swap;

CDAB=>Little-Endian Byte Swap;

DCBA=>Little-Endian

字符串长度：

LEN=1;

BCD格式：

BCD=C8421;，其中有C8421;C5421;C2421;C3;Gray

字符格式：

TEXT=UTF8;，其中有UTF8;ASCII;Default;Unicode

:::
