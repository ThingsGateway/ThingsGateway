### （一）PM2守护 

详细安装请自行查找资料

查阅详细官方文档 https://pm2.keymetrics.io/docs/usage/quick-start/

下面介绍一下常用指令

- 安装pm2

```
npm install pm2@latest -g
```


- 启用应用程序

```
pm2 start pm2-windows.json
```

- 停止应用程序

```
pm2 stop pm2-windows.json
```

- 开机自动启动

```
npm install pm2-windows-startup -g //服务安装
pm2-startup install
pm2 save //每次操作pm2应用列表时需执行
```
