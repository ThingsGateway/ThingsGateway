历史更新

ThingsGateway 框架升级/发版规则

升级前重点关注可能造成【破坏性】的标签类型：、、、

版本号规则：主版本号.次版本号.修订版本号

  • 只要【确认】为框架 bug，则当天修复，当天发版，修订版本号 加 1。
  • 如果 .csproj 文件有变更，则当天发版，修订版本号 加 1。
  • 其余情况，每年发布一个 主版本。

v1.7.0（已发布）​

  •   采集通道冗余

  •   优化多个界面

  •   导出导入功能优化

  •   增加中间变量

  •   OPCUAClient支持动态类型

  •   离线缓存多处覆盖，包含上传插件/历史报警/时序库

v1.5.0（已发布）​

  •   添加设备是否共用通道的选项（当IP&端口一致时或者串口号一致时生效）

v1.3.0（已发布）​

  •   添加IotSharp插件

v1.2.0（已发布）​

  •   添加OPCUAServer插件

v1.1.0（已发布）​

  •   添加OPCDAClient采集插件

  •   添加OPCUAClient采集插件

  •   开发串口基础框架

​

v1.0.0（已发布）​

  •   初始发布
