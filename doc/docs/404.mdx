### （一）PM2守护 

详细安装请自行查找资料

查阅详细官方文档 https://pm2.keymetrics.io/docs/usage/quick-start/

下面介绍一下常用指令

- 安装pm2

```
npm install pm2@latest -g
```

- 启用应用程序

```
pm2 start pm2-linux.json
```

- 停止应用程序

```
pm2 stop pm2-linux.json
```

- 自动启动

```
pm2-startup
pm2 save
```
